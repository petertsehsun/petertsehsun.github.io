About Myself

I am a PhD student in Computer Science at Queen's University in the Software Analysis and Intelligence Lab (SAIL). I am passionate and interested in Software Engineering. My current research focus is on improving the performance for Cloud applications that are implemented using ORM.

I also have great interests in Performance Engineering, Distributed Systems, Database, Data Mining, and Machine Learning. I am particularly interested in applying Static Analysis and Data Mining techniques to software systems to help software development.
